# 🎬 Netflix Clone - Imersão Front-End Alura 2026

Projeto desenvolvido durante a Imersão Front-End na era da IA da Alura, com o objetivo de recriar a interface da Netflix utilizando HTML, CSS e JavaScript, aplicando conceitos reais de desenvolvimento web.

---

## 🚀 Sobre o projeto

Este projeto consiste em uma aplicação web inspirada na Netflix, onde o usuário pode selecionar diferentes perfis e navegar por um catálogo de conteúdos.

A aplicação simula uma experiência real de streaming, com catálogo personalizado por perfil, controle de progresso e interações visuais modernas.

Após a finalização da imersão, o projeto foi evoluído com melhorias próprias, trazendo maior realismo e personalização.

---

## ✨ Funcionalidades

- 👤 **Seleção de perfis**  
  Escolha entre diferentes usuários ao iniciar a aplicação.

- 🎬 **Catálogo personalizado por perfil**  
  Cada perfil possui seu próprio catálogo de filmes e séries.

- 📊 **Controle de progresso**  
  Indica quanto do conteúdo já foi assistido.

- 🎥 **Preview com trailer**  
  Exibição de trailer ao passar o mouse sobre o conteúdo.

- 🌙 **Dark Mode / Light Mode**  
  Alternância de tema para melhor experiência do usuário.

- 💾 **Persistência de dados**  
  Uso de localStorage para armazenar perfil ativo.

- 🎨 **Interface inspirada na Netflix**  
  Design fiel à experiência da plataforma original.

- 🔄 **Navegação entre páginas**  
  Fluxo entre seleção de perfil e catálogo.

- ✨ **Interatividade**  
  Efeitos de hover e animações visuais.

- 📱 **Responsividade**  
  Layout adaptado para diferentes tamanhos de tela.
